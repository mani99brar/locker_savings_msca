# Creating a Smart Contract Account and sending ETH

Install dependencies:
```
npm install
```

Generate plugins:
```
npx plugingen generate
```

Get Counterfactual Address and fund wallet:
```
npm run getCFaddress
npm run sendETH
```

Install plugin:
```
npm run execute install
```

Uninstall plugin:
```
npm run execute install
```


Run index file:
```
npm run execute
```
